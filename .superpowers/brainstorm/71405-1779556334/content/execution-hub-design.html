What changes
  • /execution/ redirects to /execution/runs — no more sub-tile index page
  • Each of the 3 templates gets the module header + tab bar added at the top
  • Active tab is determined by the current route — no JS needed for navigation
  • A shared Jinja2 macro renders the tab bar so it's written once, not three times
  • Tab bar CSS added to main.css — reusable across other modules later
What stays the same
  • All 3 routes and their data-loading logic are unchanged
  • The existing content inside each template is unchanged
  • Breadcrumb in topbar stays as "Execution Hub" on all 3 pages
